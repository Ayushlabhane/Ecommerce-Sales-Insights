# 📊Ecommerce-Sales-Insights

## 📖 Project Overview
This project focuses on analyzing retail sales data to uncover revenue patterns, product performance, and regional sales trends.
The entire analysis was conducted using Microsoft Excel, starting from raw sales data and transforming it into structured insights through cleaning, aggregation, and visualization.
The goal of this project is to convert raw transactional sales data into clear, meaningful insights that help in understanding overall business performance.

### 🎯 Purpose of the Analysis
The primary purpose of this analysis is to:
- Understand overall sales performance
- Identify top-performing products and categories
- Analyze regional contribution to total sales
- Track monthly sales trends
- Present insights in a simple and business-friendly format

### 📊 Key Metrics Analyzed
- Total Sales Revenue
- Total Orders
- Total Quantity Sold
- Average Order Value
- Sales by Region
- Sales by Category and Sub-Category
- Monthly Sales Trends

### 🗂 Dataset Information
The dataset was provided in Excel format and includes:
- Order details (Order ID, Order Date, Region)
- Product details (Category, Sub-Category, Product Name)
- Sales amount
-Quantity sold
The data was cleaned and structured within Excel before performing further analysis.

### 🛠 Tools Used
- Microsoft Excel – Data cleaning, analysis, pivot tables, and dashboard creation

### 🔎 Business Questions Answered
1. What is the total revenue generated?
2. Which region contributes the highest sales?
3. What are the top 5 products based on sales?
4. Which categories generate the most revenue?
5.How do monthly sales trends vary over time?
6.Which segments or regions underperform?

### 📈 Dashboard Summary
The Excel dashboard provides:
- Sales distribution by region
- Monthly revenue trends
- Top-performing products and categories
- Key performance indicators such as total sales and order count
- The dashboard is designed to be clear and easy to interpret, allowing quick identification of trends and performance drivers.

### 📂 Project Workflow

- Imported raw sales dataset into Excel
- Cleaned and formatted data (handled inconsistencies and standardized columns)
- Created pivot tables to summarize key metrics
- Performed calculations for KPIs
- Built an interactive Excel dashboard using charts and slicers

### 📌 Key Takeaways

- Sales performance differs across regions
- A small number of products contribute significantly to total revenue
- Certain months show stronger sales trends
- Structured analysis in Excel makes business insights easier to interpret

✅ Conclusion

This project demonstrates the ability to analyze sales data using Excel and extract actionable business insights. It highlights practical skills in data cleaning, pivot table analysis, KPI calculation, and dashboard design using Excel.
